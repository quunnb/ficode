### Pinky variant

- `Ä` and `Ö` swapped

### Symbol variant

- Symbols on first layer, numbers on Shift layer

## Quickstart

- Clone repo

```sh
git clone https://github.com/quunnb/ficode && cd ficode
```

- Copy files

```sh
# cp ficode /usr/share/X11/xkb/symbols/
```

- Load keymap

```sh
setxkbmap -config ficode.conf
```
